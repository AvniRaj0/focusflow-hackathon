# FocusFlow AI

FocusFlow AI is an AI-powered productivity assistant designed to help students stay focused, prioritize their workload, and make smarter study decisions using Google Gemini.

Built for the **VIBE2SHIP Hackathon** by Coding Ninjas & Google for Developers.

---

## Features

### Smart Task Management
- Create and manage daily tasks
- Organize study workload
- Simple and distraction-free interface

###  Habit Tracker
- Track productive habits
- Monitor consistency
- Build better routines

###  AI Study Planner
Generate an intelligent daily study plan using Google Gemini.

The AI:
- Prioritizes important work
- Creates concise study goals
- Helps organize daily learning

### AI Decision Coach
FocusFlow doesn't just generate tasks—it explains its decisions.

The AI provides:
- Personalized study plan
- Why the plan was generated
- Likely avoided study task
- Confidence score
- Completion risk analysis
